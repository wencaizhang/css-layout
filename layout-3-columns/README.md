# 三栏布局

需求：假设高度已知（假设 100px），请写出三栏布局，其中左栏、右栏宽度各 300px ，中间自适应

解决方案有以下几种：

+ [浮动](./1-float.html)
+ [定位](./2-2-position.html)
+ [table-cell](./3-3-table.html)
+ [flexbox](./4-flexbox.html)
+ [grid](./5-grid.html)

请进一步思考：

1. 以上几种方案的优缺点如何
2. 如果高度未知（不固定高度），那么哪种方案依然有效？
3. 考虑到兼容性，哪种方案在业务中是最实用的，你优先选择的是哪种方案
